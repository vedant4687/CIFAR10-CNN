# 🧠 CIFAR10 CNN Image Classifier

A Convolutional Neural Network (CNN) developed using Deep Learning techniques to classify images from the CIFAR-10 dataset into ten different categories.

---

## 🎯 Features

- Image Classification
- CNN Architecture
- Training & Evaluation Pipeline
- Accuracy Visualization
- Modular Codebase

---

## 🛠️ Technologies

- Python
- TensorFlow / PyTorch
- NumPy
- Matplotlib
- OpenCV

---

## 📊 Dataset

- CIFAR-10
- 60,000 Images
- 10 Classes

---

## 🚀 Installation

```bash
git clone https://github.com/vedant4687/CIFAR10-CNN.git

cd CIFAR10-CNN

pip install -r requirements.txt
```

---

## ▶️ Run

```bash
python train.py
```
